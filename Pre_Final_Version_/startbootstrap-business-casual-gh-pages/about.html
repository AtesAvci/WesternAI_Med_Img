About Us   Our Project

This project created by Purple Rookies can accept a chest X-ray and identify if a patient has pneumonia. We have used the online dataset from: https://www.kaggle.com/paultimothymooney/chest-xray-pneumonia to diagnose the medical condition. The software used to create this project are Python and Tensorflow Keras API. Our team has worked together to create this project with very little or no prior experience in using the software tools mentioned above.
